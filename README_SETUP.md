# TravelMaker - 旅遊規劃助手

一個使用 React 開發的響應式旅遊行程規劃網站，支援多日行程管理、活動安排、預算追蹤等功能。資料使用 localStorage 儲存在本地。

## 功能特色

### ✨ 核心功能
- 🗓️ **行程建立**：選擇日期區間、地點、參與人員
- 📍 **地點建議**：智能地點關鍵字建議（例如：沖繩 → 沖繩、那霸、日本）
- 👥 **參與人員管理**：根據人數動態產生姓名輸入欄位
- 📅 **每日行程規劃**：按日期區間自動產生行程輸入介面

### 🎯 活動管理
- 🕐 **時間設定**：設定每個活動的起迄時間
- 📍 **地點搜尋**：支援地點關鍵字搜尋
- 💰 **費用追蹤**：記錄每項活動費用，自動計算總預算
- 🏷️ **分類管理**：交通、住宿、餐廳、景點、購物、其他
- 📝 **備註功能**：為每個活動添加詳細備註

### 🎨 使用者體驗
- 🔄 **拖曳排序**：使用拖放功能重新排列活動順序
- ✏️ **編輯/刪除**：輕鬆編輯或刪除已建立的活動
- 📱 **響應式設計**：完美支援桌面、平板、手機等各種裝置
- 💾 **本地儲存**：使用 localStorage 自動儲存所有資料

## 技術架構

- **前端框架**：React 18
- **建構工具**：Vite
- **拖放功能**：react-beautiful-dnd
- **日期處理**：date-fns
- **資料儲存**：localStorage
- **樣式**：純 CSS（含 CSS Variables 與 RWD）

## 安裝與執行

### 前置需求
- Node.js 16.0 或以上版本
- npm 或 yarn

### 安裝步驟

1. **安裝依賴套件**
```bash
npm install
```

2. **啟動開發伺服器**
```bash
npm run dev
```

3. **開啟瀏覽器**
預設會在 `http://localhost:5173` 啟動

### 建構生產版本

```bash
npm run build
```

建構完成的檔案會在 `dist` 資料夾中。

### 預覽生產版本

```bash
npm run preview
```

## 使用指南

### 1. 建立新行程
1. 點擊「新增行程」按鈕
2. 選擇旅遊日期區間
3. 輸入旅遊地點（會有關鍵字建議）
4. 設定參與人數與姓名
5. 點擊「建立行程」

### 2. 管理每日活動
1. 在行程詳細頁面中，每天都有「新增活動」按鈕
2. 填寫活動資訊：
   - 起迄時間
   - 活動類別
   - 地點（可選）
   - 行程內容描述
   - 費用
   - 備註（可選）
3. 點擊「儲存」

### 3. 編輯與刪除
- **編輯活動**：點擊活動卡片上的 ✏️ 圖示
- **刪除活動**：點擊活動卡片上的 🗑️ 圖示
- **調整順序**：長按並拖曳活動卡片即可重新排序

### 4. 檢視預算
- 每日費用會顯示在當天標題旁
- 總預算顯示在行程頁面頂部
