### 第一次發想 2026.02.03
1. 製作一個旅遊規劃功能的個人網站
2. 目前將焦點集中在行程新增
3. 初始化行程的資訊如下：
    1. 時間區間選擇,
    2. 地點輸入,地點輸入希望可以有關鍵字的建議,
        1. 例如：輸入沖繩 會跑出 沖繩,那霸,日本
        2. 你有更好的建議可以提出
    3. 人數,根據人數產生姓名輸入框
4. 日期與地點輸入完畢後,有一個按鈕會產生日期區間的行程輸入介面
5. 可以輸入的資訊有：
    1. 可以搜尋地點,帶出google map簡易資訊,
    2. 起迄時間
    3. 行程內容
    4. 費用
    5. 行程類別
    6. 備註
6. 每一天的行程如果有輸入地點,希望介面上有google map帶出路線圖,以及路線圖上要有這是哪裡的資訊
7. 行程之間需要可以拖曳,刪除與編輯
